# Teste

* Para testar, basta rodar:
    * Versão Java: `java Calculadora.java`
    * Versão C: 
        * Compilar: `gcc calculadora.c -o calculadora_c`
        * Executar: `./calculadora_c`
    * Versão Haskell:
        * Compilar: `ghc -o calculadora_hs calculadora.hs`
        * Executar: `./calculadora`
