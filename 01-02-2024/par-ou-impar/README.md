# Teste

* Para testar, basta rodar:
    * Versão Java: `java ParOuImpar.java`
    * Versão C: 
        * Compilar: `gcc par_ou_impar.c -o par_ou_impar_c`
        * Executar: `./par_ou_impar_c`
    * Versão Haskell:
        * Compilar: `ghc -o par_ou_impar_hs par_ou_impar.hs`
        * Executar: `./par_ou_impar_hs`
