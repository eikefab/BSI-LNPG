# Teste

* Para testar, basta rodar:
    * Versão Java: `java Maioridade.java`
    * Versão C: 
        * Compilar: `gcc maioridade.c -o maioridade_c`
        * Executar: `./maioridade_c`
    * Versão Haskell:
        * Compilar: `ghc -o maioridade_hs maioridade.hs`
        * Executar: `./maioridade_hs`
