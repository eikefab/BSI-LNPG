# Teste

* Para testar, basta rodar:
    * Versão Java: `java Vestibular.java`
    * Versão C: 
        * Compilar: `gcc vestibular.c -o vestibular_c` 
        * Executar: `./vestibular_c`
    * Versão Haskell:
        * Compilar: `ghc -o vestibular_hs vestibular.hs`
        * Executar: `./vestibular_hs`
