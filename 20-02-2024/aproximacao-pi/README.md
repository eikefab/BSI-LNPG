# Teste

* Para testar, basta rodar:
    * Versão Java: `java Pi.java`
    * Versão C: 
        * Compilar: `gcc pi.c -o pi_c -lm`
          * `-lm` necessário pois utilizo o módulo `math`.
        * Executar: `./pi_c`
    * Versão Haskell:
        * Compilar: `ghc -o pi_hs pi.hs`
        * Executar: `./pi_hs`
